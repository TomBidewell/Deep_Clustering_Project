# Deep_Clustering_Project

This notebook contains the code for my Bachelor's Thesis. In it I propose a new deep embedding technique based on AutoEncoders and Kullback Leibler (KL) divergence that. Current state of the art methods create a probability distribution, Q, over the clusters using each point's distance to the centre of that cluster; an ideal distribution, P, is then made as a function of Q and the KL loss between the two is used to update parameters. The method I propose creates a "pseudo" ideal distribution by using the already trained KMeans algorithm to predict cluster labels for each of the data points. This method significantly improves the clusters produced, shown by much better silhouette and inertia scores for this projects data. This project is still ongoing and so the notebook is not fully finished.
